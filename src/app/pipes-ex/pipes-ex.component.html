Pipes

Pipes in angular is use to format the data using pre defined pipes which is present in Common Module

We can write custom pipes to fulfill requirements


{{myDate | date :'yyyy-MM-dd'}}
{{name | lowercase }}
{{course | uppercase }}
{{myDate | date}}
{{name | slice : 3:7 }}
{{name| nametitle :"male"}}
{{"Harley"| nametitle :"female"}}
